# Ecommerce-Sales-Analysis
Data visualization and sales analysis using Python and matplotlib
Developers Arena - Data Science Internship  
Author : Sudharshan Katakam

---

## Project Overview
This project is part of Data Visualization.
The goal of this project is to perform a complete data analysis workflow, starting from loading and cleaning a real-world dataset to analyzing it and visualizing key insights.
The project uses Python, pandas, matplotlib, and Jupyter Notebook to analyze an e-commerce sales dataset.
By combining numerical analysis with visualizations, the project demonstrates how data can be transformed into meaningful insights that support better understanding and decision-making.

---

## Project Objective
- Load and explore a real-world sales dataset  
- Clean the dataset by handling missing values and duplicates  
- Perform basic sales analysis  
- Create at least **two different types of charts**  
- Generate meaningful insights from the analysis  
- Present results with professional documentation

---

## Dataset Used
This dataset contains e-commerce sales information used for data analysis and visualization.
- Product – Name of the product sold
- Price – Price of a single unit
- Customer_ID – Unique identifier for each customer
- Region – Sales region
- Total_Sales – Total sales amount for each transaction
